# Product Overview

不動産物件をユーザー定義のルールベースで自動採点する個人用Webアプリ（Property Evaluator）。
AIの曖昧な回答や主観的判断に頼らず、一貫したルールベースの定量評価を実現する。

## Core Capabilities

1. **プロフィール管理**: 年収・貯蓄・金利などの個人財務情報を登録
2. **ルールベース評価**: ユーザー定義の閾値・スコアリングルールで物件を自動採点
3. **カテゴリ重み付け**: 資金適合性(50%)、リセールポテンシャル(30%)、建物スペック(20%)の3軸評価
4. **物件比較**: スコアに基づく物件のランキングと比較

## Target Use Cases

- 複数物件を同一基準で定量評価したい個人の不動産購入検討者
- MVP段階では開発者自身が唯一のユーザー（シングルユーザー前提）
- 物件情報を手動入力し、自分の優先度に基づいたスコアで意思決定を支援

## Value Proposition

- **一貫性**: 毎回同じ基準で評価（AIの回答ブレを排除）
- **財務整合性**: 個人の財務プロファイルに基づいた現実的な評価
- **カスタマイズ性**: 自分にとって重要な指標・重みを自由に設定

## Scope Boundaries (MVP)

- URL自動取得（SUUMO、HOME'S等）は対象外
- AIコメント生成は対象外
- マルチユーザー認証は対象外
- 市場データの自動連携は対象外
